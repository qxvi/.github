# qxvi Public Boundary Interface

qxvi is the public visual/interface layer for machine-action boundary visibility.

It exists to make one question legible:

What crossed the boundary?

BOUNDARYCAM is the primary qxvi surface. It is a public product/control surface with routes, schemas, examples, manifest, validation, release rail, and GitHub Pages deployment.

qxvi does not claim evidence authority by itself.

qxvi shows the boundary frame.

Evidence recording belongs to INVOCORDER.

Verification and completion perimeter belong to VERIFRAX.

Public registry/control routing belongs to SPEEDKIT.
